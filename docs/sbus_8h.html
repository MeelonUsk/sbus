uint8_t SBus_ParseByte ( uint8_t  byte)

Parses currently read byte into frame.

Adds the passed byte into the SBus frame. If after the byte the frame is built, function returns SBUS_FRAME_READY. Otherwise, function returns SBUS_FRAME_NOT_READY.
